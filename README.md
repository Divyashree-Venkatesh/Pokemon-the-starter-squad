# Pokemon-the-starter-squad
🎨 The Starter Squad: An Artist’s Entry into Frontend
Welcome to The Starter Squad! This project marks my official "Level 1" as a Frontend Developer. Transitioning from a background in traditional drawing and art, I built this interactive Pokémon selection interface to practice translating visual concepts into clean, functional code.

🌟 Overview
The Starter Squad is a responsive web interface that allows users to explore and "pick" their starting team. My goal was to move away from static canvases and explore how HTML and CSS can create a dynamic user experience.

🛠️ Built With
This project is built using pure, frontend technologies to ensure a strong grasp of the fundamentals:

HTML5: Structured for accessibility and semantic clarity.

CSS3: Custom styling featuring:

Flexbox & Grid: Used for a perfectly aligned, sketchbook-style layout.

CSS Transitions: Smooth hover effects to bring the "cards" to life.

Media Queries: Ensuring the squad looks great on any screen size.

🎨 The "Art-to-Code" Process
Coming from a drawing background, I approached this project with a focus on:

Composition: Balancing the characters within their cards.

Color Theory: Using the Pokémon's elemental types (Water, Grass, Electric) to dictate the UI color palette.

Attention to Detail: Ensuring the spacing (padding/margin) is as precise as a finished illustration.
